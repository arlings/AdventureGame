Summary: This is a game made without the use of classes or any related simplifications, simply grade 10 programming knowledge and some pygame. This includes methods.
Controls: Use WASD to move around. Left click to place items. Left click to attack. Use the crafting menu to craft specific items.
Bugs/Missing Parts: Zombies cannot move when they must move vertically to get closer to the player. This leads to zombies freezing up. 
When the exterior of the map is reached, there is nothing stopping the player from going over the edge. 
How to play: Download the src folder, and run this using an IDE. Make sure you have python and pygame installed. 
Install the Pygame library itself using Python's package manager (run pip install pygame). Open the python file.
Make sure that when the program is running, it is in the same folder as all of the image files to ensure a smooth run, and that all images are accessible.
